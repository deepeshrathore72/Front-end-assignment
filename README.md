# React App - IT Health Check

A React application for Wendigo Advisors IT Health Check service.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

- `/src` - Source code
  - `/components` - React components
  - `/lib` - Utility functions
  - `App.jsx` - Main app component
  - `main.jsx` - Entry point
  - `index.css` - Global styles
